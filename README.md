# SAE32_2023

Réalisé par Maxime LELONG, Kohsey DUFOUR et Axel PIETROIS

## Suppression

Pour supprimer l'archive .jar :
```properties
make clean
```

## Compilation

Pour générer l'archive .jar lancez la commande suivante :
```properties
make build
```

## Execution

Pour lancer le tableur lancez la commande suivante :
```properties
make tableur
```

Pour tout lancer :
```properties
make all
```
